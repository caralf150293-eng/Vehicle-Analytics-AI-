# 📌 Descripción

Vehicle Analytics AI es una aplicación desarrollada en Python y Streamlit que permite analizar un conjunto de datos de una agencia de automóviles mediante técnicas de análisis exploratorio y Machine Learning.

La aplicación integra:

- Dashboard Ejecutivo
- Visualizaciones Interactivas
- Matriz de Correlación
- Modelo Predictivo
- Predicción de vehículos
- Reporte Ejecutivo
